# 🌈 קשת חברים — מדריך העלאה ל-GitHub Pages

## למה GitHub Pages?
- ✅ **חינם לחלוטין**
- ✅ **HTTPS אוטומטי** — חובה לקול ומיקרופון!
- ✅ **עובד על אנדרואיד ו-iOS**
- ✅ **קישור ציבורי** שאפשר לשתף

---

## שלב 1️⃣ — צור חשבון GitHub (פעם אחת)

1. כנס ל: https://github.com/signup
2. הרשם בחינם — צריך רק אימייל וסיסמה

---

## שלב 2️⃣ — צור Repository חדש

1. לחץ על **"+"** למעלה משמאל → **"New repository"**
2. שם: `keshet-chaveirim` (או כל שם אחר)
3. סמן **Public**
4. סמן ✅ **"Add a README file"**
5. לחץ **"Create repository"**

---

## שלב 3️⃣ — העלה את הקובץ index.html

### דרך 1: גרירה ושחרור (הכי קל!)
1. ב-repository החדש שלך — לחץ **"Add file"** → **"Upload files"**
2. **גרור את הקובץ `index.html`** לחלון הדפדפן
3. למטה — לחץ **"Commit changes"**

### דרך 2: דרך הממשק
1. לחץ **"Add file"** → **"Create new file"**
2. שם הקובץ: `index.html`
3. הדבק את התוכן של `index.html`
4. למטה — לחץ **"Commit new file"**

---

## שלב 4️⃣ — הפעל GitHub Pages

1. ב-repository — לחץ על **Settings** (למעלה)
2. בתפריט השמאלי — **Pages**
3. תחת **"Build and deployment"**:
   - **Source**: Deploy from a branch
   - **Branch**: `main` ו-`/ (root)`
4. לחץ **"Save"**
5. המתן 1-2 דקות...

---

## שלב 5️⃣ — קבל את הכתובת!

תופיע הודעה:
```
✅ Your site is live at:
https://YOUR-USERNAME.github.io/keshet-chaveirim/
```

📱 **פתח את הקישור הזה בטלפון/מחשב** וזהו!

---

## 📲 איך להתקין על הטלפון?

### אנדרואיד (Chrome):
1. פתח את הקישור
2. לחץ על **⋮** (3 נקודות)
3. **"Install app"** → **"Install"**

### iOS (Safari):
1. פתח את הקישור
2. לחץ על **🔗** (Share)
3. **"Add to Home Screen"**

האפליקציה תופיע במסך הבית כמו אפליקציה רגילה! 🎉

---

## 🔧 אם הקול לא עובד באנדרואיד:

1. **התקן מ-Play Store**: "Speech Services by Google"
2. **הגדרות מערכת**: 
   - הגדרות → ניהול כללי → שפה והזנה
   - → טקסט-לדיבור → "מנוע דיבור Google"
3. **התקן עברית**: 🔧 → התקן נתוני קול → עברית
4. רענן את הדף

---

## 🆘 בעיות נפוצות

| בעיה | פתרון |
|------|--------|
| "Site not found" | המתן 2-3 דקות אחרי הפעלת Pages |
| הקול לא עובד | ודא שזה https:// בכתובת |
| המיקרופון לא נפתח | אפשר הרשאה בהגדרות הדפדפן |
| העמוד לבן | בדוק ש-index.html בשורש (לא בתיקייה) |

---

## 🎁 בונוס: עדכון האפליקציה

כשאתה רוצה לעדכן:
1. ב-repository → לחץ על `index.html`
2. לחץ על אייקון העפרון ✏️
3. ערוך → **Commit changes**
4. תוך דקה האתר מתעדכן אוטומטית!

---

## 📞 צריך עזרה?

אם משהו לא עובד — פתח GitHub Issue ב-repository או שלח אימייל.
